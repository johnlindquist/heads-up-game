Heads Up Game Admin Guide

Security Note: This version uses a public Gist to store categories, which is more secure as it doesn't require storing any credentials.

How to Manage Categories

Initial Setup

  1. Go to GitHub Gist
  2. Create a new file called categories.json
  3. Copy this template and modify it with your categories: 
    {
    "animals": {
        "name": "Animals",
        "words": ["Lion", "Elephant", "Giraffe"]
    },
    "movies": {
        "name": "Movies",
        "words": ["Avatar", "Star Wars", "Matrix"]
    }
}
  4. Click "Create public gist"
  5. Copy the Gist ID from the URL (the long string after your username)
  6. Update the GIST_ID in script.js with your Gist ID

Updating Categories

  1. Go to your Gist on GitHub
  2. Click the "Edit" button
  3. Modify the categories and words as needed
  4. Click "Update public gist"
  5. The game will automatically load the new categories when players refresh

Category Format

Each category in the JSON file should follow this format:

{
    "category-id": {
        "name": "Display Name",
        "words": ["Word 1", "Word 2", "Word 3"]
    }
}

Tips

  • Keep category IDs lowercase and simple (e.g., "movies", "animals")
  • Display names can be more user-friendly (e.g., "Popular Movies", "Wild Animals")
  • Make sure your JSON is valid - you can use a JSON validator if needed
  • The game will fall back to the local categories.json if the Gist fails to load
